Installation Guide

Follow these simple steps to install the Listo Chrome extension

1

Download the Extension

Click the "Download Extension" button above to download the ZIP file containing the extension files.

2

Extract the Files

Extract the downloaded ZIP file to a folder on your computer. You'll see files like manifest.json, popup.html, and other extension files.

3

Open Chrome Extensions

Open Google Chrome and navigate to chrome://extensions/ in the address bar, or go to Chrome Menu → More Tools → Extensions.

4

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page to enable installation of unpacked extensions.

5

Load the Extension

Click "Load unpacked" and select the folder containing the extracted extension files. The Listo extension should now appear in your extensions list.

6

Pin the Extension

Click the puzzle piece icon in Chrome's toolbar and pin the Listo extension for easy access. You're all set!
